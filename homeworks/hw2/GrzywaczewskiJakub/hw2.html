Wizualizacja

choices <- c("Prawo i Sprawiedliwo", 
             "Koalicja Obywatelska",
             "Polska 2050",
             "Lewica",
             "Poslkie Stronnictwo Ludowe",
             "Konfederacja",
             "niezdecydowani")
values <- c(0.33, 0.267, 0.096, 0.093, 0.062, 0.052, 0.10) * 100
colors <- c("#405c99", "#db214e", "#fcbd16", "#95135d", "#00a64e", "#18304e", "#f0ac66")

df <- data.frame(choices, values, colors)
df <- df[match(choices, df$choices), ]
df$choices <- fct_rev(factor(df$choices, levels = df$choices))

df %>% 
  ggplot(aes(x = choices, y = values, fill = colors)) +
  geom_bar(stat = "identity", fill = colors) +
  coord_flip() +
  labs(
    title = "Aktualne poparcie dla ugrupowa politycznych",
    y = "Poparcie (wynik sondy)",
    x = "Ugrupowanie polityczne"
  ) +
  theme(
    plot.margin = margin(20,20,20,10),
    legend.margin = margin(5, 10, 5, 10)
  )+
  theme_bw() +
  geom_text(aes(label = paste(values, "%", sep="")), hjust = -0.3, colour = "black") +
  scale_y_continuous(expand = c(0, 0), limits = c(0, 100))

Uzasadnienie

Postcie s nie do skali wzgldem odestku poparcia

Uycie postaci reprezentantw danych partii politycznych jako supkw, o ile kreatywne, nie przynosi niestety odpowiedniego rezultatu ze wzgldu na ludzk fizjologie. Jeeli chcielibymy zachowa odpowiednie proporcje wysokoci danej postaci do procentu poparcia mielibymy mae i postacie, ktre nie przensioyby odpowiedniej informacji.

Prosty wykres supkowy

W przedstawionej powyej poprawionej wizualizacji postawiem na prosty wykres supkowy z dobrze opisanymi osiami, wynikami sondy oraz odpowiednio wizualizujcy skal rnicy gosw. Jako dodatek supki s pokolorowane kolorami kluczowymi dla danej partii co pozwoli ogldajcemu lepiej odnale si w caym wykresie.
